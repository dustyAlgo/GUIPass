# Random Password Generator

A simple and user-friendly random password generator built with Python and Tkinter. This application allows users to generate secure passwords based on specified criteria, including password length and strength.

## Features

- **Password Strength Options**: Choose between low, medium, and strong password strengths.
- **Customizable Length**: Select the desired length of the generated password.
- **Clipboard Support**: Easily copy the generated password to your clipboard for convenient use.

## Requirements

- Python 3.x
- Tkinter (comes pre-installed with Python)
- `pyperclip` for clipboard functionality
